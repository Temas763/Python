# Проект автоматизации тестирования

## Описание
Этот проект содержит автоматизированные UI и API тесты sky.pro . 
Проект создан в рамках итогового задания курса "Автоматизация тестирования".

## Структура проекта

## Структура проекта
project/
├── config/          # Конфигурация приложения
├── data/            # Тестовые данные и настройки
├── pages/           # Page Objects для UI тестов
├── tests/           # Тесты (UI и API)
├── utils/           # Вспомогательные утилиты
├── .env             # Конфиденциальные данные
├── pytest.ini       # Конфигурация pytest
├── README.md        # Документация
└── requirements.txt # Зависимости Python

## Предварительные требования
- Python 3.9+
- Установленный браузер Chrome

## Установка и запуск

1. Клонируйте репозиторий:

2. Установите зависимости:
pip install -r requirements.txt
3. Запуск тестов:
pytest tests/ # закуск всех тестов
pytest tests/test_api.py # закуск тестов api
pytest tests/test_ui.py # закуск тестов ui
allure serve allure-results # Генерация отчета Allure
