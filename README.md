# Проект автоматизации тестирования

## Описание
Этот проект содержит автоматизированные UI и API тесты для веб-приложения [Название приложения]. 
Проект создан в рамках итогового задания курса "Автоматизация тестирования".

## Структура проекта

## Структура проекта
project/
├── tests/ # Тесты (UI и API)
├── pages/ # Page Objects для UI тестов
├── api/ # API клиенты
├── config/ # Конфигурация
├── utils/ # Вспомогательные утилиты
└── requirements.txt # Зависимости

## Предварительные требования
- Python 3.9+
- Установленный браузер Chrome

## Установка и запуск

1. Клонируйте репозиторий:

2. Установите зависимости:
pip install -r requirements.txt
3. Запуск тестов:
pytest tests/
